SoloWeightsWeb
Impound just got a whole lot easier...and more accurate!

SoloWeights is a minimum weight calculator developed by an autocrosser for autocrossers. Initially conceived as an Android app, this cross-platform web version is mobile ready and is designed to work on any device that has a web browser!
